# Cybersecurity-Jovac
Project of Jovac on OWASP TOP 5 using Docker
## Overview
This project provides a containerized lab environment that integrates the OWASP Top 5 web vulnerabilities using Docker. It is designed to help developers, testers, and security enthusiasts understand, simulate, and mitigate the most critical security risks in modern web applications..
## What It Does
- Deploys vulnerable web applications in isolated Docker containers.
- Demonstrates real-world exploitation scenarios for:
  - Broken Access Control
  - Security Misconfiguration
  - Software Supply Chain Failures
  - Cryptographic Failures
  - Injection Attacks
- Offers guided walkthroughs and remediation strategies.
- Enables safe, repeatable testing without affecting production systems.
